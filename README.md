# Blog

Personal blogging application.

## Development

This project needs to have `node.js`, `docker` and `docker-compose` installed on the development machine.

To start development database instance run:

```shell
npm run db:dev
```

To start development server with code watch and automatic reload run:

```shell
npm run start:dev
```
